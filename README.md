# ЁЯУМ Project Overview

The **Movie Industry Performance Analysis** project is an end-to-end Data Analytics portfolio project that explores financial and audience performance across the movie industry. Using **Microsoft Excel** for data cleaning and **Power BI** for interactive visualization, this project transforms raw movie data into actionable business insights.

The dashboard enables stakeholders to evaluate revenue, budget, profitability, IMDb ratings, and language-wise performance, supporting data-driven decision-making.

---

# ЁЯОп Business Problem Statement

Movie production involves substantial financial investments, but not every film achieves commercial success. Production companies, investors, and business stakeholders need to understand how factors such as production budget, language, industry, and audience ratings influence a movie's financial performance.

This project analyzes movie industry data to identify the key drivers of success, measure profitability, and uncover trends that support strategic investment decisions.

---

# ЁЯОп Project Objective

The objective of this project is to:

- Analyze movie industry performance
- Compare production budget and revenue
- Measure Return on Investment (ROI)
- Evaluate IMDb ratings
- Compare industry-wise and language-wise performance
- Identify high-performing movies
- Build an interactive Power BI dashboard for business decision-making

# ЁЯУМ Key Business Insights

- Bollywood generated the highest overall revenue.
- Higher production budgets generally resulted in higher box office collections.
- English-language movies achieved stronger average IMDb ratings.
- Revenue distribution varies significantly across industries.
- ROI analysis highlights the most profitable movies regardless of production budget.

---

# ЁЯТб Business Recommendations

- Invest in industries with consistently higher ROI.
- Optimize production budgets based on historical performance.
- Prioritize high-performing language markets.
- Use audience ratings alongside financial metrics for investment decisions.
- Monitor ROI rather than revenue alone when evaluating project success.
